# The Kitchen

A static menu site for Bernhard & Marna's recomp + BP-aware meal plan.

- **Live:** https://copilot-clive.github.io/our-kitchen/
- **Daily target:** 2,114 kcal / 180 C / 70 F / 191 P
- **Goal:** 80 kg / 15% BF by 1 Dec 2026
- **Layered for:** DASH-style blood-pressure (sodium ≤1,500 mg, K+ ≥4,000 mg, daily nitrate, no aged cheeses)

Single-file static site (`index.html`), no dependencies, no build. Designed mobile-first.

Built 2026-05-17.
